# CD_DataLake
Datalake of Olympics data from 1896 to 2024

## Sources
* https://www.kaggle.com/datasets/piterfm/paris-2024-olympic-summer-games
* https://basedosdados.org/dataset/62f8cb83-ac37-48be-874b-b94dd92d3e2b

## Raw
Raw .csv files acquired from the sources. Inside the parquet folder, they have been converted to .parquet files

## Processed
Processed .csv data from the raw files:
* Athletes per gender
* Medals per country
* Modalities

## Analytics
Analysis of the processed data
